155  * a whole frame.
156  *
157  * @param[in] tdp pointer to a @p MACTransmitDescriptor structure
158  * @param[in] size size of the requested buffer. Specify the frame size
159  * on the first call then scale the value down subtracting
160  * the amount of data already copied into the previous
161  * buffers.
162  * @param[out] sizep pointer to variable receiving the real buffer size.
163  * The returned value can be less than the amount
164  * requested, this means that more buffers must be
165  * requested in order to fill the frame data entirely.
166  * @return Pointer to the returned buffer.
167  *
168  * @api

171  mac_lld_get_next_transmit_buffer(tdp, size, sizep)

173 /**

177  * frame.
178  *
