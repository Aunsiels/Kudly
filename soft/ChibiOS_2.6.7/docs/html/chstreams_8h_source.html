28 /**
29  * @file chstreams.h
30  * @brief Data streams.
31  * @details This header defines abstract interfaces useful to access generic
32  * data streams in a standardized way.
33  *
34  * @addtogroup data_streams
35  * @details This module define an abstract interface for generic data streams.
36  * Note that no code is present, just abstract interfaces-like
37  * structures, you should look at the system as to a set of
38  * abstract C++ classes (even if written in C). This system
39  * has then advantage to make the access to data streams
40  * independent from the implementation logic.<br>
41  * The stream interface can be used as base class for high level
42  * object types such as files, sockets, serial ports, pipes etc.
43  * @{
44  */
