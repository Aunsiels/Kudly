109  * @note Multiple Event Listeners can use the same event identifier, the
110  * listener will share the callback function.
111  *
112  * @param[in] esp pointer to the @p EventSource structure
113  * @param[out] elp pointer to the @p EventListener structure
114  * @param[in] eid numeric identifier assigned to the Event Listener. The
