28 /**
29  * @file chfiles.h
30  * @brief Data files.
31  * @details This header defines abstract interfaces useful to access generic
32  * data files in a standardized way.
33  *
34  * @addtogroup data_files
35  * @details This module define an abstract interface for generic data files by
36  * extending the @p BaseSequentialStream interface. Note that no code
37  * is present, data files are just abstract interface-like structures,
38  * you should look at the systems as to a set of abstract C++ classes
39  * (even if written in C). This system has the advantage to make the
40  * access to streams independent from the implementation logic.<br>
41  * The data files interface can be used as base class for high level
42  * object types such as an API for a File System implementation.
43  * @{
44  */
